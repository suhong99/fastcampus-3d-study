# React + Vite

R3F는 three.js와 달리 react 친화적이고, 사용이 간단함

### gltfjsx

gld를 jsx로 만들어 커스텀 할 수 있게 해줌

```bash
npx gltfjsx public/dancer.glb -o src/components/Dancer.jsx
```

### PostProcessing

후처리를 위해 사용

```bash
npm i @react-three/postprocessing
```

### use-cannon

물리엔진 처리에 사용

```bash
npm i @react-three/cannon
```
